Curso Git Github

comando do git mais importantes

  • git init inicializa o repositório local
  • git add . adiciona a stage area um arquivo do repoitório
  • git commit -m "mais comentario" comita o arquivo"
  • git push sobe os comites para o repositório do Github
  • git pull busca o que já estava no repositorio do github

Como subir arquivos para o Github

  1. git branch -M main Cria a branch no github
  2. o comonado endereço do seu repositório no github| git remote add origin https://github.com/joseneisantana/curso_ebac_frontend.git
  3. git push -u origin main chama a origem

    4. Tudo isso é feito depois de um repositório criado no git local em seu pc
